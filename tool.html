🎣 URL Phishing Detector

Analyze a URL for common phishing red flags. This is a heuristic tool — always use caution.

🧠 What to look for in phishing URLs:
• Misspelled brand names (paypa1.com, g00gle.com)
• Suspicious TLDs (.xyz, .tk, .cc, .pw)
• Hyphens in domain (paypal-secure-login.com)
• IP address instead of domain (http://192.168.1.1/login)
• Excessive subdomains (login.paypal.com.evil.com)
• No HTTPS
• URL shorteners hiding real destination
